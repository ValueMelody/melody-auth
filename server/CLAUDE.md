# Server

## Swagger

`server/src/scripts/swagger.json` is autogenerated — never edit it directly.

To update the S2S API spec:
1. Update the schema source in `server/src/scripts/schemas/*.cjs` (e.g. `user.cjs` for User/UserDetail shapes)
2. Update the JSDoc `@swagger` comment in the relevant route file under `server/src/routes/` for request/response shapes specific to an endpoint
3. Run the generate script from the project root:

```
npm run s2s:generate
```

This regenerates `swagger.json` and also regenerates `admin-panel/services/auth/api.ts` from the updated spec.
