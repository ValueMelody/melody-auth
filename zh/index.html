Melody Auth

Effortless OAuth and Authentication, Built for Flexibility and Control.

GitHub截图演示更新日志

灵活的部署选项

在 Cloudflare 的全球边缘网络中部署，实现低延迟的性能，或者选择自托管 Node.js 环境，搭配 Redis 和 PostgreSQL 以获得对基础设施的最大控制。

全面的 OAuth & 认证系统

利用 OAuth 2.0 的强大功能，以及多因素认证、暴力破解保护、基于角色的访问控制和无缝的用户授权流程等高级功能。

开发者友好的工具

通过用户友好的管理面板、轻松的前端集成和安全的服务器间通信的 REST API 简化你的工作流程。
