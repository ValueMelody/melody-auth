# Rules

When adding, updating, or removing a doc file that is linked in the VuePress config (`docs/.vuepress/config.js`), always keep the corresponding Chinese version under `docs/zh/` in sync.

When adding a new feature doc, also update the root `README.md` to reflect the feature (e.g. under the relevant section in "Auth Server Features Supported" or "Admin Panel & S2S REST API Features Supported").
